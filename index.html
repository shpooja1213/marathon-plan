Google Sheets setup

Step 1 — paste your Apps Script URL

Once you've deployed your Google Apps Script (see instructions below), paste the web app URL here. All your data will sync automatically every time you make a change.


Step 2 — set up Google Sheets & Apps Script

Follow these steps exactly. It takes about 5 minutes.

  1. Go to sheets.google.com and create a new spreadsheet. Name it Pooja Marathon Plan.
  2. Create these 4 sheets (tabs at the bottom): Checklist, Weights, Runs, Lifts
  3. In the spreadsheet, click Extensions → Apps Script
  4. Delete all existing code and paste the Apps Script code below (copy it all).
  5. Click Save (disk icon), then click Deploy → New deployment
  6. Set type to Web app. Set "Execute as" to Me. Set "Who has access" to Anyone. Click Deploy.
  7. Copy the web app URL and paste it above in Step 1.
